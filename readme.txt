MCQS Developer is a professional WordPress plugin to manage Multiple Choice Questions, Exams, Analytics, and User Attempts.

== Description ==

MCQS Developer provides a powerful platform for educators, trainers, and organizations to manage MCQ banks, create exams, analyze results, and moreâ€”all within WordPress.

**Features:**
- MCQ question bank management
- Categories for questions
- Exam/quiz creation
- User dashboard showing profile and quiz/exam attempt history
- Admin analytics (questions, categories, exams)
- Import/export tools (stub, extendable)
- Two widgets (MCQS stats, recent items)
- Developer-friendly modular code

== Installation ==

1. Upload the `mcqs-developer` folder to the `/wp-content/plugins/` directory, or upload the ZIP via Plugins > Add New > Upload Plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the admin menu for MCQ management, exam creation, analytics, and import/export.
4. Add the `[mcqs_user_dashboard]` shortcode to any page for users to view their dashboard.

== Frequently Asked Questions ==

= How do users see their quiz/exam history? =
Add the `[mcqs_user_dashboard]` shortcode to any page.

= Can I extend this plugin? =
Yes! All logic is modular and resides in `/includes`. Add new features by creating new modules and including them.

= Is there analytics? =
Yes, see the Analytics page in the admin menu.
